## 教程
本方案建议**有一定搞机基础的人员**使用（仅适用于**安卓手机**且**仅限电脑**使用）    
如何使用：   
1. 将你的手机连接到电脑    
2. 下载仓库里的反和谐文件（根据系统下载）   
3. 解压后运行`Replace.py`   
4. 打开蔚蓝档案
